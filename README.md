FebriZip Gateway (OPC Server / Data Bridge PLC & Microcontrollers)
- 
Fungsi :
- Sebagai OPC Server 
- Sebagai Data Bridge / Gateway 

Fitur Dari Aplikasi Ini :
- Membaca Data Dari PLC & Microcontroller Seperti Arduino
- Banyak Jenis Protocol Yang bisa digunakan untuk membaca data dari PLC
- Read Data PLC Langsung Melalui Fitur Tag Configurator (Menyesuaikan Address PLC / Microcontroller)
- Sebagai Routing Menuju Node Red Secara Instant. Mengirim Data PLC Ke Node-Red / MQTT Secara Praktis
- Buku Panduan Untuk Pengguna.

Support Controller
-
- PLC (Mitsubishi , Siemens, Omron, Scheneider, Allen Bradley, Dll)
- Microcontroller (NodeMCU / WEMOS / ESP8266 / Arduino)
